Laravel Model Observers
Installation and use

$ git clone https://github.com/mobiosolutions/laravel-observer-example.git
$ cp .env.example .env

Change configuration according to your need in ".env" file and create Database

$ composer install
$ php artisan migrate
$ php artisan serve
